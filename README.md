# Mandelbrot

![Preview](preview.png "Preview")

## Controls
- `W` - Move up
- `A` - Move down
- `S` - Move left
- `D` - Move right
- `Up` - Zoom in
- `Down` - Zoom out
- `Left` - Decrease iterations
- `Right` - Increase iterations
- `R` - Reset to default state
- `1` - Point of interest 1
- `2` - Point of interest 2
- `Esc` - Exit
